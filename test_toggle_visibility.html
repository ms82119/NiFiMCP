How to see this in the real app:

  1. Start a new conversation with a request that will trigger tool calls
  2. Wait for the workflow to complete (you'll see live status updates)
  3. Look for the final message that contains the execution summary
  4. The toggle button will appear above the execution steps in that final message

Note: The toggle button only appears in the final aggregated message after a workflow completes, not in the live status updates during execution.
